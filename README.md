## Risk-weighted impact (RWI)

This repository includes a python notebook with core functions for RWI calculation (and referenced data). 

For details on the concept of RWI, see rwimpact.com or peer-reviewed publication in Neurosurgery (https://journals.lww.com/neurosurgery/fulltext/9900/risk_weighted_impact__reframing_risk_analyses_for.2137.aspx).
